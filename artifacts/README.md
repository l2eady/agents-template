# Artifacts Protocol
This directory serves as the **Permanent Memory** and **Active Workspace** for the AI Agent.

## Folder Structure

- **`/plans`**: Implementation plans generated BEFORE coding starts.
  - Naming: `plan_[id or task_name].md`
- **`/logs`**: Raw output from tests, terminal commands, or debugging sessions.
  - Naming: `log_[date]_[description].txt`
- **`/research`**: Raw material, findings, comparisons, and initial "brain dumps".
  - Naming: `[topic].md`

> **Rule:** If you are "thinking", put it here. Do not clutter the chat context.
